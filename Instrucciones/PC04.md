## 📋 Objetivo

Tomar como base el código completo de la **PC03** (PostalMundial API con todos los métodos implementados) y crear una **colección de Postman** que permita ejecutar sin problemas todos los endpoints disponibles. Organizar el repositorio siguiendo la estructura de carpetas requerida.

---

## 🎯 Competencias a Evaluar

- Pruebas manuales de APIs REST con Postman
- Organización de repositorio con separación código / pruebas
- Cobertura de todos los métodos HTTP del API
- Verificación de flujo completo CRUD + consulta de faltantes

---

## 🏗️ Estructura Requerida del Repositorio

El repositorio **debe** respetar exactamente la siguiente estructura:

```
📁 codigo/
│   └── (código fuente completo de la PC03 — solución .sln y proyectos)
📁 pruebas/
│   └── Album.json          ← colección de Postman
README.md
```

> ⚠️ **Importante**: No se aceptarán entregas con estructura diferente.  
> El código fuente va en `codigo/` y la colección de Postman en `pruebas/`.

---

## 📝 Tareas a Completar

### ✅ 1. Copiar código de la PC03

Copiar el código fuente completo de la PC03 dentro de la carpeta `codigo/`.  
El proyecto debe **compilar sin errores** desde esa carpeta.

### ✅ 2. Crear la colección de Postman

Crear una colección de Postman llamada **`Album`** que incluya una request para cada uno de los siguientes endpoints:

| # | Método | Endpoint | Descripción |
|---|--------|----------|-------------|
| 1 | `GET` | `/api/postal` | Obtener todas las postales |
| 2 | `GET` | `/api/postal/faltantes` | Obtener postales faltantes (`Tengo = 0`) |
| 3 | `GET` | `/api/postal/{id}` | Obtener una postal por ID |
| 4 | `POST` | `/api/postal` | Agregar nueva postal |
| 5 | `PUT` | `/api/postal/{id}` | Editar postal existente |
| 6 | `DELETE` | `/api/postal/{id}` | Eliminar postal |

Cada request debe:
- Tener un nombre descriptivo (ej. `Obtener todas las postales`)
- Incluir el body JSON correspondiente para los métodos `POST` y `PUT`
- Apuntar a `http://localhost:{puerto}/api/postal`

**Ejemplo de body para POST/PUT:**
```json
{
  "numero": 5,
  "posicion": "Delantera",
  "tengo": true,
  "notas": "Postal en buen estado",
  "idPais": "00000000-0000-0000-0000-000000000000",
  "idGrupo": "00000000-0000-0000-0000-000000000000"
}
```

### ✅ 3. Exportar la colección

Exportar la colección desde Postman en formato **Collection v2.1 (JSON)** y guardarla como:

```
pruebas/Album.json
```

### ✅ 4. Subir al repositorio

Hacer commit y push con la siguiente estructura completa:

```
codigo/   ← código fuente
pruebas/  ← Album.json
README.md
```
